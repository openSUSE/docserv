Chapter 1. Terminology

build instruction

An instruction to build (or rebuild) a specific set of documents in specific language.

build server

The machine that runs Docserv². This machine can but does not have to be different from the target.

docset

A set of documents, usually represents a specific version of a product.

Git source repository

A Git repository that holds the source code of a document/set of documents.

site section

A homepage-level category.

target

The machine that serves built documentation output. This machine can but does not have to be different from the build server.
