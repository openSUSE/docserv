Chapter 4. Configuration overview

Configuration of Docserv² is split in a number of parts that are described in the following.

  • Site configuration configures the site and sync targets in itself. For more information, see Chapter 5, Site configuration.

  • Product configuration configures what documentation to publish. For more information, see Chapter 6, Product configuration.

  • Templates for navigational pages configure the look of the homepage and product overview pages. For more information, see Chapter 7, Navigational page templates.

  • Localization files allow localizing the homepage and product overview pages. For more information, see Chapter 8, Localization.

  • SSI fragments allow reusing HTML page fragments as part of different navigational pages. For more information, see Chapter 9, SSI fragments.

The order above gives a rough idea of the order and importance of each item, though you are free to work on them any other order.
